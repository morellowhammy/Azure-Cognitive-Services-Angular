Exercise 1: Cognitive Services - Face API

In this first part of the workshop we will be looking into Microsoft Cognitive Services. First of, we will be using the Face API, where we will post an image of a person to a REST-api and get back a result containing information related to the persons emotions, as perceived by the AI.

Notice: This exercise requires node >= 6.9 and npm. If you don't have this already installed. Download and install it first node and npm.

Objectives

  1. Obtain your Face API Key
  2. Take a selfie and handle the result, displaying the emoji that best reflects the emotion returned by the Face API (e.g. angry emoji for anger)

1. Setup

  1. Download the following repository for this exercise: ML Workshop.
  2. Got to Azure Cognitive Services.
  3. Select Face APIs and click the Get API Key button.
  4. Agree to the terms and click next.
  5. If prompted, sign in with your account.
  6. A popup will welcome you and let you know you have 7 days to try out the Face APIs for free.
  7. Click the "Got it" button
  8. You should now be presented with 2 keys and a URL. Copy one of the keys and the url and paste it into the src/app/shared/face.service.ts file in the Angular source folder, replacing the subscriptionKey and uriBase.

Now that you have configured your API key we are ready to try out some artificial intelligence.

2. Run the application.

  • Navigate to the base folder in a terminal and type ng serve -o
  • This should install the required packages and run the web site on localhost:4200
  • Allow the site to use web camera.
  • Capture a selfie and watch the results.

4. Code

You can search the TODOs in the application code to find the places you need to code in.

The exercise is to parse Face API result and show the correct smiley for the emotion. Look at the shoot-face.component.ts and emoji.service.ts files to get started.

Good luck!
